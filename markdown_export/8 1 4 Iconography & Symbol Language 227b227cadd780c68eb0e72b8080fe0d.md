# 8.1.4 Iconography & Symbol Language

| Symbol Type | Design Direction |
| --- | --- |
| **Magic Glyphs** | Geometric + ancient: triangles, spirals, interlocking loops |
| **Skill Icons** | Abstract and organic, like cave paintings or tattoos |
| **UI Markers** | Faint etchings, glowing softly—not glowing neon or high-contrast |
| **Language** | Optional symbolic runes with a decipherable logic (for tomes, ruins) |

All icons should feel **ancient yet grounded**, as if carved by real people with spiritual intent.
