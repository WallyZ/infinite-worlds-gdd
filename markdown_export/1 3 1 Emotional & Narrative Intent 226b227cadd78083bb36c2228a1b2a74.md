# 1.3.1 Emotional & Narrative Intent

| Emotion | System Support |
| --- | --- |
| **Presence & Significance** | No quick-saves; every decision matters |
| **Connection to Others** | NPC memory systems, cultural memory |
| **Bittersweet Reflection** | Legacy mechanics, personal artifacts |
| **Inspiration for New Characters** | World shifts, inherited quests, ancestry visions |

Death becomes a **passing of the torch** — not a game over.
