# 1.11.3.2 Practice as Reclamation

Practicing a skill is framed as **refining what is already known**, not learning from zero. The more a player performs a task:

- The more **precise**, **fluid**, and **confident** the animations and interactions become.
- Subtle **techniques or variations** begin to manifest (e.g., quicker carving strokes, instinctive parries, smoother incantations).
- The game provides **behavioral reinforcement**, not UI meters.

Players aren’t told *“You’ve leveled up swordsmanship”*, but feel it through changes like:

- Less stamina loss on swings.
- Cleaner follow-through on attacks.
- NPCs reacting with increasing respect or curiosity.
