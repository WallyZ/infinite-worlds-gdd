# 1.11.3.5 Sensory and Embodied Feedback (VR Emphasis)

In VR, skills become increasingly embodied through:

- **Physical movement fidelity**: If the character is trained in swordplay, correct swings are subtly "helped" (haptic reinforcement, muscle memory guides).
- **Feedback cues**: Incorrect movements feel sluggish, while correct ones feel natural or energized.
- **Environmental responses**: Objects respond more intuitively when handled correctly—knives bite deeper, spells stabilize faster, etc.

Over time, this creates a **non-verbal feedback loop** where the *player feels the character’s competence*.
