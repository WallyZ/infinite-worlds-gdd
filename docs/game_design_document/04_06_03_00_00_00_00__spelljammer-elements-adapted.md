# 4.6.3 Spelljammer Elements Adapted

- **Spelljamming Helm Equivalent** → "Star Throne" or "Resonance Core"
    - Activated by ritual or seated meditation
    - Reads player’s mind/gestures and translates to directional momentum
- **Air Bubbles** → “Soulfield Envelopes”
    - Generated by resonance between pilot, runes, and core
    - Degrades over time or if ship integrity is lost
- **Phlogiston / Wildspace** → “The Aether Veil”
    - Luminous realm between worlds, pulsing with intention and memory
    - Thought, music, and fear affect navigation
    - Storms and anomalies are metaphysical, not weather-based (e.g., zones of grief or euphoria)
- **Crystalline Spheres** → “Worldshells”
    - Every planet or world floats in a spiritual membrane, only passable via:
        - Astral alignment
        - Sacred phrase
        - Planetary permission (NPC factions, divine rites)
