# 15.1 Glossary

Common terms, acronyms, gameplay shorthand, metaphysical concepts

*A quick-reference dictionary of key terms used throughout the GDD to ensure clarity across team members and stakeholders.*

| Term | Definition |
| --- | --- |
| **VR** | Virtual Reality — A fully immersive digital experience using head-mounted displays and motion-tracked input. |
| **XP (Experience Points)** | A metric used to track player progression and skill development. |
| **IAP** | In-App Purchase — Microtransactions made within the game environment. |
| **Blueprints** | Unreal Engine's visual scripting system. Used for prototyping and building gameplay systems without C++. |
| **Tactile Feedback** | Haptic responses used in VR to simulate the feeling of touch or impact. |
| **NPC** | Non-Player Character — Characters controlled by AI to interact with the player or world. |
| **Hard/Soft Currency** | Soft currency is easily earned through gameplay; hard currency is typically purchased with real money. |
| **Crafting Tier System** | A progression-based crafting structure where complexity and accuracy increase as skills improve. |
