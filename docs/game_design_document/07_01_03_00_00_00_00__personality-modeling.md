# 7.1.3 Personality Modeling

###

[🧠](data:image/gif;base64,R0lGODlhAQABAIAAAP///wAAACH5BAEAAAAALAAAAAABAAEAAAICRAEAOw==)

Each NPC has a set of **core traits** (e.g., honesty, loyalty, boldness, greed) which influence how they react to the world and one another.

Emotional states are **transient modifiers** (e.g., anger, sorrow, joy) that color current behavior.

Chemistry and compatibility determine the **likelihood of friendship, romance, or rivalry**, including with other NPCs.

A stoic knight and mischievous thief may despise each other at first but warm over time — unless the player’s favoritism fuels resentment.

- Trait-based system (e.g., stubborn, generous, cautious)
- Personality changes through trauma, aging, rituals, social influence
