# 7.4.1.1 Navigation Mesh Integration

- Unreal generates a **NavMesh** dynamically across traversable surfaces.
- NPCs use this mesh to calculate paths in real time.
- If an object is marked as "Dynamic Obstacle," the NavMesh can update around it automatically.
- **You do not need to predetermine exact paths** — only the environment layout and behavior of obstacles matter.
