# 1.11.2.1 Skill Familiarity Through Exposure

When the player first encounters tools, environments, or practices their character is already familiar with, the game uses **subtle cues and behavioral feedback** to reinforce that sense of competence or discomfort:

- **Correct grip, stance, posture** animations play for skills the character knows well.
- Objects subtly **highlight, glint, or draw attention** if the character would instinctively recognize them (e.g., poisonous plants, structural weaknesses).
- **Interaction prompts** reflect confidence or hesitation:
    - “You move as if you’ve done this before…”
    - “Your fingers fumble—this is not your craft.”
