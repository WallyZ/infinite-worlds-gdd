# 8.1.5 Tone of Voice & Text

| Element | Style |
| --- | --- |
| **Narration** | Poetic, mythic, understated. "The land breathes," not "It has dynamic weather." |
| **UI Text** | Minimalist. “You are hungry.” / “The wind whispers.” |
| **Marketing Copy** | Reflective, emotional, symbolic. Avoid “grind, gear, win” language. |
| **Player Messaging** | Encourage curiosity, responsibility, and self-directed action. |

Use the style of *Dark Souls item descriptions*, *Outer Wilds* lore logs, or *Tunic*’s manual—**evocative, not explanatory**.
