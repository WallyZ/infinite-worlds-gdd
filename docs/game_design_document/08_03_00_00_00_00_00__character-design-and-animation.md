# 8.3 Character Design & Animation

### 🧍 Character Design & Animation

**🧬 Humans & Humanoids:**

- Anatomically correct human base meshes
- Cultural variation by region, class, profession, and alignment
- Realistic clothing physics and layers (armor over padding, belts, pouches, accessories)

**🧙‍♂️ Fantasy Creatures:**

- Creatures grounded in plausible biology: muscle groups, load-bearing structures, respiration, etc.
- Hybrids (e.g., gryphons, fae) designed with anatomical logic—no “floating” or non-functional body parts
- Use of AI tools to assist in concept variation, evolution, and ecological placement

**🎞️ Animation Systems:**

- Full-body **motion capture** for grounded locomotion, gestures, posture shifts
- Procedural locomotion blending (turning, walking on inclines, speed shifts)
- **Contextual animation sets**: foraging, climbing, combat stances, spell casting
- VR-specific body IK and hand presence, supporting gesture casting and interaction fidelity
- Integration of **AI-assisted retargeting** (MetaHuman Animator, DeepMotion) for variation across races and builds

**🎞️ Animation Techniques:**

- Full-body mocap, VR-based IK, procedural movement blending

**🧰 Tools for Character Art & Animation:**

| Task | Free Tools | Paid Tools |
| --- | --- | --- |
| Character Modeling | Blender, MakeHuman | ZBrush, Maya, 3ds Max |
| Clothing Simulation | Blender (Cloth sim), Marvelous Designer (trial) | Marvelous Designer |
| Character Rigging | Auto-Rig Pro (free lite), Mixamo | AccuRIG, Maya Rigging Tools |
| Motion Capture | DeepMotion (basic free tier), Mixamo | Rokoko Studio, iPi Soft, Plask, Noitom Perception Neuron |
| Retargeting & Cleanup | Blender, Unreal Engine’s built-in tools | MetaHuman Animator, DeepMotion Pro |
| Facial Animation | Face Cap (iOS), Unreal Live Link | Dynamixyz, Faceware Studio |
