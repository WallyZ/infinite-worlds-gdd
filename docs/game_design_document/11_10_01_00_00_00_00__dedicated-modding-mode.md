# 11.10.1 Dedicated Modding Mode

- Full sandbox with built-in testing environments
- Visual scripting tools & AI editor

To foster creativity, safety, and ease of development, *Infinite Worlds* offers a **Dedicated Modding Mode** — a sandboxed environment for building and testing new content without risk to persistent saves, multiplayer balance, or canonical world data.

This mode is both a **developer toolset** and an **immersive creation space** for lore-conscious modders.

---

### 🔓 **Modes of Access**

- **Launcher Toggle:**
    - Select *Modding Mode* from game launcher or platform UI (e.g., Steam)
    - Opens separate instance of the game with full debug overlays, asset hot-reloading, and isolated memory
- **In-World Entry (Lore-Compatible Option):**
    - Accessible via special in-game artifacts (e.g., “Scribe’s Tablet”, “Mirror of Realities”) in sandbox worlds
    - Opens diegetic UIs for item scripting, region painting, or entity editing
    - Used for immersive building of magic, history, and environment-friendly mods

---

### 🧱 **Core Features**

- **Safe Testing Environment:**
    - All simulation systems run as they would in the main game
    - NPC behaviors, AI memory, physics, and crafting can be verified before deployment
    - Debug commands and simulation speed adjustment available
- **Hot-Reloading & Runtime Editing:**
    - Modify assets, scripts, or blueprints and see changes in real time
    - World changes (e.g., terrain sculpting, AI logic, inventory injection) reflected immediately
- **Integrated Modding Panels:**
    - **World Editor**: Adjust terrain, add structures, define POIs
    - **Faction/AI Editor**: Define new factions, AI routines, emotional matrices, and diplomacy logic
    - **Spell & Skill Editor**: Use templates to build and customize rituals, gestures, or psionics
    - **Dialogue/Story Editor**: Write branching NPC stories, plug into existing world events or create your own

---

### 🌐 **Multiplayer-Aware Modding**

- **Multiplayer Sandbox Testing:**
    - Host a temporary modded server with friends or test players
    - Simulate NPC gossip spread, cross-player narrative propagation, and modded cooperative play
- **Mod Permissions Settings:**
    - Choose whether mods are:
        - Private (local use only)
        - Server-specific (for private worlds)
        - Public-compatible (with version and compatibility checks)

---

### 🧙 **Diegetic Creation Tools (Optional Mode)**

- **Lore-Compatible Modding UI** (toggleable for RP creators):
    - Use in-game objects like:
        - **Runes of Alteration** – create spells or tweak magical systems
        - **Ancestral Forge** – craft items, weapons, or blueprints
        - **Dream Table** – alter ecosystems or regional geography
    - Suitable for streamers, narrative-driven creators, or modders who want a fully immersive dev experience

---

### 🔁 **Deployment Workflow**

1. **Build/Test in Modding Mode**
2. **Export as .pak or plugin module**
3. **Auto-generate metadata (name, dependencies, version, tags)**
4. **Deploy to:**
    - Local install
    - Steam Workshop / Nexus Mods
    - Server-specific repository
    - Private mod packs

---

### 🧠 **Advanced Developer Options**

- **Blueprint Debugger**
    - Step-by-step event and tick analysis for scripted systems
- **Custom Simulation Tools**
    - Trigger long-term NPC or ecological simulations without waiting in real time
- **Performance Profiler Overlay**
    - Track memory, CPU, GPU, and replication load for modded assets

---

### 📖 **Documentation & Templates**

- **Included Templates:**
    - Blank spell, NPC, building, biome, and item templates
    - Blueprint macro libraries and reusable functions
    - Dialogue and behavior tree starter packs
- **Example Mods Included:**
    - Learning tools showing best practices, optimization strategies, and multiplayer-safe coding
- **In-Mode Docs Panel**
    - Quick access to API references, dev notes, and simulation logic documentation

---

This *Dedicated Modding Mode* is a cornerstone for building a **vibrant, creator-led community**, where mods can be lore-rich, technically sound, and deeply tied to the procedural world’s logic.
